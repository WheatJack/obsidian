## 2. 数据流

```
cxcx.iachina.cn
       │
       │ POST /queryProductResult.html (分页)
       │ GET  /viewProduct.html?code=xxx (详情)
       │ GET  /pages/.../ViewContext.jsp (条款/费率)
       ▼
 Python 爬虫 (cxcx_crawler.py)
       │
       │ GET  /insurance/product/exists (检查存在)
       │ POST /insurance/product/saveOrUpdate (保存)
       ▼
 MVP 服务 (Spring Boot :8088)
       │
       ▼
 MySQL (insurance_product 表)
```

## 3. 目标网站接口分析

### 3.1 产品列表查询

- **URL**: `POST https://cxcx.iachina.cn/queryProductResult.html`
- **编码**: GBK
- **认证**: Cookie (JSESSIONID)
- **分页**: `pageNo` 参数, 每页 10 条

**请求参数:**

```
pageNo=1
pageCount=0
equipmentType=01
queryProductInfo.prodRegNo=
queryProductInfo.prodName=
comName1=
queryProductInfo.salesStatus=01     # 01=在售, 02=停售
queryProductInfo.prodDevType=309    # 产品类别代码
queryProductInfo.coverageType=      # 1=主险, 2=附加险
```

**约束**: 注册号/产品名称/公司名称/产品类别至少填一项

### 3.2 产品详情

- **URL**: `GET https://cxcx.iachina.cn/viewProduct.html?code={uuid}&t={timestamp}`
- **返回**: HTML 页面，包含注册时间、停售时间、条款/费率文件链接

### 3.3 条款/费率下载

- **URL**: `GET https://cxcx.iachina.cn/pages/iacpcprodinfo/ViewContext.jsp`
- **参数**: `filename={uuid}.html&productId={id}&productNo={regNo}&flag=tiaokuan|feilv`

## 4. 爬虫策略

### 4.1 采集策略

遍历 **19个产品类别 × 2种状态** = 38个查询组合，每个组合分页爬取全部数据。

### 4.2 去重策略

以**注册号 (regNo)** 为唯一标识：
1. 每条数据保存前先调用 `GET /insurance/product/exists?regNo=xxx`
2. 已存在则跳过
3. 不存在则获取详情后保存

### 4.3 防封策略

- 请求间隔 0.3 秒
- 使用标准浏览器 User-Agent
- 携带完整 Cookie

### 4.4 Session 管理

- 首次由人工登录获取 cookies，通过 API 传入系统
- 每分钟由 MVP 定时任务检测 session 有效性
- 失效时邮件通知（每4小时限频一次）
- 收到通知后人工重新登录获取 cookies，调用 API 更新

## 5. API 设计

### 5.1 产品数据接口

**判断产品是否存在**
```
GET /insurance/product/exists?regNo=C000012134022026040298443
Response: {"code": 200, "data": true}
```

**保存/更新产品**
```
POST /insurance/product/saveOrUpdate
Body: {
  "companyName": "中华联合财产保险股份有限公司",
  "productName": "...",

  "coverageType": "附加险",
  "productCategory": "医疗意外保险",
```

### 5.2 爬虫管理接口

**更新 Cookies**
```
POST /insurance/crawler/updateCookies
Body: {"cookies": "JSESSIONID=xxx;lasttime=yyy"}
Response: {"code": 200, "data": "Cookies 更新成功"}
```

**触发爬虫（可选传入cookies）**
```
POST /insurance/crawler/start
Body: {"cookies": "JSESSIONID=xxx"}  // 可选
Response: {"code": 200, "data": "爬虫任务已启动"}
```

**查看状态**
```
GET /insurance/crawler/status
Response: {
  "code": 200,
  "data": {
    "running": false,
    "lastRunStatus": "完成 - 新增150/跳过8000/失败0",
    "cookiesConfigured": true
  }
}
```
