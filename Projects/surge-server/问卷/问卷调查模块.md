# 问卷调查模块

## 概述

问卷调查模块支持在后台 Admin 创建问卷，通过 App 分享给微信客户填写，或 App 用户直接填写。

## 涉及项目

| 端 | 项目 | 说明 |
|---|------|------|
| Admin 后台 | surge-admin | 问卷创建、编辑、上下架、统计 |
| App 端 | surge-app | 问卷列表、详情、填写、分享 |
| 微信 H5 端 | surge-wechat | 客户填写问卷 |
| 后端服务 | surge-mvp | 问卷 CRUD、提交、统计接口 |
| 传播链服务 | blade-instrument | 传播链追踪、消息通知 |
