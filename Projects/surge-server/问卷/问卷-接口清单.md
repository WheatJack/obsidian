# 接口清单

## Admin 接口 `/admin/questionnaire`

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /save | 保存问卷(整体保存含题目+选项) |
| GET | /page | 分页查询(按tenantId隔离) |
| GET | /detail/{id} | 问卷详情(含题目+选项) |
| POST | /publish/{id} | 上架 |
| POST | /unpublish/{id} | 下架 |
| DELETE | /delete/{id} | 删除(已上架不可删) |
| POST | /copy/{id} | 复制 |
| GET | /share/{id} | 获取分享信息(链接+二维码) |
| GET | /stats/{id} | 统计分析 |
| GET | /submissions/{id} | 答卷列表 |
| GET | /submission/detail/{id} | 答卷详情 |

## App 接口 `/app/questionnaire`

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /list | 查询当前用户可见的问卷列表 |
| GET | /{id} | 问卷详情 |

> [!important] tenantId
> App列表接口从 `SecureUtil.getTenantId()` 自动获取，不依赖前端传参。
> 拿不到直接返回null。

## 微信/通用接口 `/wechat/questionnaire`

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /{id} | 问卷详情(填写渲染用) |
| GET | /check/{id} | 检查是否已提交(支持openId/userId) |
| POST | /submit | 提交答卷 |

> [!tip] check接口优化
> check接口直接调 `getSubmittedAnswers`，非空即已提交。
> 一次查询同时返回 `submitted` + `answers`，避免查两次submission表。
