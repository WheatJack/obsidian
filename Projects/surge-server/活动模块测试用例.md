## 一、测试方案

### 1.1 测试分层

| 层级 | 工具 | 覆盖范围 |
|------|------|---------|
| 单元测试 | JUnit5 + Mockito | Service 层业务逻辑 |
| Controller 测试 | JUnit5 + Mockito | 接口入参出参、状态码 |
| 接口集成测试 | REST Assured / HTTP 脚本 | 真实 API 端到端调用 |
| 前端 E2E 测试 | Cypress / Playwright | 页面交互流程 |

### 1.2 运行方式

```bash
# 单元测试 + Controller 测试
mvn test -pl surge-mvp -Dtest="com.surge.activity.*"

# 接口集成测试（需启动服务）
mvn test -pl surge-mvp -Dtest="com.surge.activity.integration.*"
```
