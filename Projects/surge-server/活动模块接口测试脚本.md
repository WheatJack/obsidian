# 活动模块接口测试脚本

> **使用方式**: IntelliJ IDEA HTTP Client 或 VS Code REST Client  
> **修改变量**: 修改 `@host` 和 `@activityId` 为实际值  
> **同步来源**: `surge-server/surge-mvp/doc/activity-api-test.http`

---

## 环境变量

```http
@host = https://surge-api-test.insbean.com/api
@activityId = 2034980273027837953
@userId = 2008922189247574018
```

---

## A. 活动列表

### B-01 活动列表-默认分页
```http
GET {{host}}/surge-mvp/app/activity/list?current=1&size=10
```

### B-07 活动列表-筛选进行中
```http
GET {{host}}/surge-mvp/app/activity/list?current=1&size=10&status=2
```

### B-08 活动列表-筛选待开始
```http
GET {{host}}/surge-mvp/app/activity/list?current=1&size=10&status=1
```

### B-09 活动列表-筛选已结束
```http
GET {{host}}/surge-mvp/app/activity/list?current=1&size=10&status=3
```

### B-10 活动列表-按关键字搜索
```http
GET {{host}}/surge-mvp/app/activity/list?current=1&size=10&keyword=发布会
```
