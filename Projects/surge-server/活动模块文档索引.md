## 🎯 关键信息速查

### 数据库表

| 表名 | 用途 |
|------|------|
| `surge_activity` | 活动主表 |
| `surge_activity_form_field` | 表单字段配置表 |
| `surge_user_activity` | 用户活动参与表 |

### 活动状态

| 状态码 | 状态 | 说明 |
|--------|------|------|
| 0 | 草稿 | 默认状态，不对外展示 |
| 1 | 已发布/待开始 | 已发布但时间未到 |
| 2 | 进行中 | 当前时间在活动时间内 |
| 3 | 已结束 | 当前时间超过结束时间 |
| 4 | 已取消 | 手动取消，不可修改 |

### Material Type

```
MATERIAL_TYPE_ACTIVITY = '1'  // 活动
MATERIAL_TYPE_QUESTIONNAIRE = '5'  // 问卷
```

### 消息 Flag

| Flag | 触发场景 | 来源服务 |
|------|---------|---------|
| `view_article` | 客户浏览活动 | blade-instrument |
| `share_article` | 客户转发活动 | blade-instrument |
| `activity_register` | 客户报名活动 | surge-mvp |
| `activity_checkin` | 客户签到活动 | surge-mvp |
